# Sawubona, World! 🇿🇦

A South African-themed Hello World page. *Sawubona* is a Zulu greeting meaning "I see you."

## Features

- South African flag color palette (green, gold, black, red, blue)
- GSAP entrance animations
- Responsive typography
- Accessible (respects `prefers-reduced-motion`)

## Live

View it live: [Sawubona World on Hyperagent](https://pub.hyperagent.com/p/ciIEulaivLoVKmP5h-i8wg?v=2)

## Deploy

Just serve `index.html` — it's a single static file with no build step. Works on GitHub Pages, Vercel, Netlify, or any static host.

To enable GitHub Pages: **Settings → Pages → Source: main branch → / (root)**
