# Sawubona, World! 🇿🇦

A South African-themed Hello World page with cinematic animations. *Sawubona* is a Zulu greeting meaning "I see you."

## Features

- Full-bleed Cape Town / Table Mountain background photo
- Ken Burns slow zoom + parallax on mouse movement
- Golden light shimmer + floating particle effects
- Animated SVG ocean waves
- South African flag color palette (green, gold, black, red, blue)
- GSAP entrance animations
- Responsive typography
- Accessible (respects `prefers-reduced-motion`)

## Deploy

Just serve `index.html` and `cape-town-bg.png` — no build step needed.

To enable GitHub Pages: **Settings → Pages → Source: main branch → / (root)**
